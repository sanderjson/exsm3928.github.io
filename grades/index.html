Grades Overview
EXSM 3928 Code for Designers

Grades

Grades are made up of three components: discussions, coding challenges, and quizzes. Each module has a discussion and a coding challenge. There are four quizzes - one Module 2, Module 3, Module 4, and Module 5.

Discussions

There are seven discussions, one for each module. Each discussion is scored out of eight points. To complete each discussion you must post an original response to the assignment prompt AND you must respond to one original post from another student. Each of your compositions is scored out of four points:

  • (1) point for posting on eClass
  • (1) point for adding meaningful content to the discussion
  • (1) point for explaining WHY you chose a given discussion
  • (1) point for writing with proper grammar and spelling

Coding Challenges

There are seven coding challenges, one for each module. Each challenge consists of a TODO list. Each item that is complete gets one mark. Each coding challenge is unique.

Coding Challenge 6 and 7 have additional scoring rubriks attached to their respective assignments. These challenges require more effort, and have greater value.

Quizzes (see eClass)

There are four quizzes each with ten questions. Each question is weighted equally. You have unlimited attempts at the quizzes. Quiz questions are randomly ordered.

  • Website knowledge (Module 2)
  • HTML (Module 3)
  • CSS (Module 4)
  • JavaScript (Module 5)
